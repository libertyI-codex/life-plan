# 家計ライフプラン表 ローカル版 Ver.1.2 試作3

Ver.1.2 試作3では、毎年使い続けるための計算設定と、ライフプラン表の表示改善を追加しました。

## Ver.1.2 試作3で追加した内容

- 計算設定セクションを追加しました。
- 計算開始年を変更できるようにしました。
- 計算終了年を変更できるようにしました。
- 現在資産の基準年を設定できるようにしました。
- メモを入力できるようにしました。
- 毎年、基準年と現在資産・収入・支出を更新して使い続けられる設計にしました。
- ライフプラン表の列グループを表示・非表示できるようにしました。
- ライフプラン表のコンパクト表示に対応しました。

## 追加したデータ構造

- `settings.baseYear`
- `settings.memo`

旧データにこれらがない場合は `ensureDataShape()` で自動補完します。

## 追加したUI状態

- `uiState.tableColumns`
- `uiState.compactLifePlanTable`

保存先は以下です。

`lifePlanLocal.uiState.v1`

## GitHub Pages 公開版

- GitHub Pages公開版を Ver.1.2 試作3相当に更新しました。
- 計算設定セクションを追加しています。
- 計算開始年・計算終了年・現在資産の基準年を変更できます。
- 毎年、基準年と現在資産・収入・支出を更新して使い続けられる設計です。
- ライフプラン表の列グループを表示・非表示できます。
- ライフプラン表のコンパクト表示に対応しています。
- ドル建生命保険は総資産外の参考資産として扱います。
- 家計データ保存キーは `lifePlanLocal.data.v1` のまま変更していません。
- UI状態保存キーは `lifePlanLocal.uiState.v1` を継続使用しています。
- GitHub Pages更新前にはJSONバックアップを取ってください。

## 表示列設定

ライフプラン表では、以下の列グループを表示・非表示できます。

- 家族年齢
- 収入
- 通常支出
- イベント支出
- 年間収支
- 年末資産
- 住宅ローン
- 純資産
- ドル建生命保険

## コンパクト表示

コンパクト表示では、表の文字サイズとセル余白を小さくします。

また、円表示を表示上だけ短縮します。

- 1,000,000円 → 100万円
- 12,500,000円 → 1,250万円
- 0円 → 0

内部データと計算値は変更しません。

## 継続している重要仕様

- ドル建生命保険は総資産外の参考資産として扱います。
- ドル建生命保険の解約返戻金は総資産・純資産・年末資産見込みには含めません。
- 住宅ローン残高推移は維持しています。
- バックアップ文字列表示・コピーは維持しています。
- JSONエクスポート・インポートは維持しています。
- セクション折りたたみは維持しています。
