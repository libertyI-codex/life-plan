# 家計ライフプラン表 ローカル版 Ver.1.3 試作1

- ドル建終身保険の解約返戻金推移機能を追加しました。
- 米ドル建ての解約返戻金を円換算できます。
- 想定為替レートを入力できます。
- キャッシュバリュー表をCSV風テキストで入力できます。
- ライフプラン表に参考列として表示します。
- 今回の試作では総資産・純資産には自動反映していません。
- 二重計上防止のため、自動反映は次回以降に検討します。
- 保存キー `lifePlanLocal.data.v1` は変更していません。
- UI状態保存キー `lifePlanLocal.uiState.v1` を継続使用しています。

## ドル建終身保険の扱い

- ドル建終身保険の解約返戻金は総資産には含めない方針です。
- 純資産・年末資産見込みにも含めません。
- アプリ上では総資産外の参考資産として扱います。
- 二重計上防止、為替変動、流動性、税金未計算の観点からこの扱いにしています。
- 将来的に総資産へ自動反映する予定は現時点ではありません。

## Ver.1.3 試作2

- 2024年以降を対象とするNISA枠管理機能を追加しました。
- 本人・配偶者の2人分を分けて管理できます。
- 各管理枠の対象者は家族設定から選択し、`familyMemberId` で家族情報に追従します。
- 同じ家族を2つのNISA管理枠へ重複登録することはできません。
- つみたて投資枠と成長投資枠を分けて管理します。
- 年別につみたて・成長の買付額、売却簿価、メモを入力できます。
- 年間投資枠は、つみたて120万円、成長240万円、合計360万円として残り額を計算します。
- 年末のつみたて・成長・合計の非課税保有簿価残高を計算します。
- 生涯総枠1,800万円、成長投資枠1,200万円として翌年利用可能額を計算します。
- 当年の売却簿価合計を翌年復活予定額として表示します。
- 売却しても、その年の年間投資枠は戻りません。
- 売却簿価には売却代金ではなく、NISAで取得したときの金額を入力します。
- 2023年以前の旧NISAは対象外です。
- NISA枠管理の買付額、売却簿価、年末簿価残高、残り枠は総資産へ加算しません。
- NISA枠管理は税制上の利用枠を把握する独立した参考機能です。証券会社の表示を優先してください。
- 旧データに `nisaManagement` がない場合は `ensureDataShape()` で2人分の初期構造を補完します。
- 補完処理を複数回実行しても管理枠や年次記録は重複追加しません。
- JSONエクスポート・インポートに `nisaManagement` を含めます。
- 家計データ保存キー `lifePlanLocal.data.v1` は変更していません。
- UI状態保存キー `lifePlanLocal.uiState.v1` は変更していません。
- GitHub Pages公開前にはJSONエクスポート、またはバックアップ文字列コピーでバックアップを取ってください。
