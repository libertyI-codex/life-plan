# 家計ライフプラン表 ローカル版 Ver.1.1 試作1

- 家計ライフプラン表 ローカル版 Ver.1.1 試作1です。
- Ver.1.0正式版に住宅ローン残高推移機能を追加しました。
- 住宅ローン現在残高、金利、毎月返済額を入力できます。
- 年ごとの住宅ローン残高を概算計算できます。
- ライフプラン表に住宅ローン年初残高、住宅ローン年末残高、住宅ローン年間利息、純資産見込みを追加しました。
- ダッシュボードに現在住宅ローン残高、最終年の住宅ローン残高、住宅ローン完済予定年、最終年の純資産見込みを追加しました。
- 住宅ローン残高は概算であり、金融機関の返済予定表とは一致しない場合があります。
- 既存データに `mortgage` がない場合も読み込み時に自動補完します。
- 古いJSONバックアップも読み込めます。
- localStorage保存キーは `lifePlanLocal.data.v1` で変更していません。
- 繰上返済、ボーナス返済、住宅ローン控除は未実装です。
- 税金、社会保険料、NISA/iDeCo利回り、厳密な誕生日判定は未実装です。
