# 家計ライフプラン表 ローカル版 Ver.1.2 試作4

- ドル建生命保険の解約返戻金推移機能を追加しました。
- 米ドル建ての解約返戻金を円換算できます。
- 想定為替レートを入力できます。
- キャッシュバリュー表をCSV風テキストで入力できます。
- ライフプラン表に参考列として表示します。
- 今回の試作では総資産・純資産には自動反映していません。
- 二重計上防止のため、自動反映は次回以降に検討します。
- 保存キー `lifePlanLocal.data.v1` は変更していません。
- UI状態保存キー `lifePlanLocal.uiState.v1` を継続使用しています。

## ドル建生命保険の扱い

- ドル建生命保険の解約返戻金は総資産には含めない方針です。
- 純資産・年末資産見込みにも含めません。
- アプリ上では総資産外の参考資産として扱います。
- 二重計上防止、為替変動、流動性、税金未計算の観点からこの扱いにしています。
- 将来的に総資産へ自動反映する予定は現時点ではありません。
