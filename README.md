# 家計ライフプラン表 ローカル版 Ver.1.0 正式版候補

## 目的

家族の年齢、人生イベント、現在資産、収入、支出をもとに、年単位で将来の資産推移を確認するローカルWebアプリです。

## 実装済み機能

- ダッシュボード表示
- 現在総資産、現在純資産、最終年の資産見込み、資産マイナス年、大きな支出イベント数の表示
- 家族追加・編集・削除
- 家族の名前、生年月日、続柄、メモの管理
- 関連イベントがある家族の削除ブロック
- イベント追加・編集・削除
- イベント名、カテゴリ、対象者、予定年、対象者年齢、金額、メモの管理
- 資産・収入・支出の編集
- 現金・預金、投資資産、保険解約返戻金、その他資産、負債の入力
- 手取り年収、賞与、手当、その他収入の入力
- 生活費、住宅ローン、保険、教育費、車関連、その他固定費の月額入力
- 年単位ライフプラン表
- 家族ごとの年齢列表示
- イベント支出、年間収入、通常支出、年間収支、年初資産、年末資産の表示
- 年末資産グラフ
- localStorage保存
- 初期データリセット
- JSONエクスポート
- JSONインポート

## 保存仕様

localStorage保存キー：

`lifePlanLocal.data.v1`

## JSONバックアップ

- JSONエクスポートで `life-plan-backup-YYYY-MM-DD.json` を保存できます。
- JSONインポートでバックアップデータを復元できます。
- インポート前に確認ダイアログを表示します。
- 不正JSONや構造不備の場合はエラー表示します。

## 計算仕様

- 金額は手取りベース入力を前提とします。
- 現在総資産 = 現金・預金 + 投資資産 + 保険解約返戻金 + その他資産
- 現在純資産 = 現在総資産 - 負債
- 年間収入 = 手取り年収 + 賞与 + 手当 + その他収入
- 通常年間支出 = 毎月支出合計 × 12
- 年間支出 = 通常年間支出 + イベント支出
- 年間収支 = 年間収入 - 年間支出
- 年末資産 = 年初資産 + 年間収支
- 負債は純資産表示に使用し、年末資産推移にはまだ直接反映しません。

## 注意点

- 税金の詳細自動計算は未実装です。
- 社会保険料の詳細自動計算は未実装です。
- 住宅ローン残高の年次減少計算は未実装です。
- 厳密な誕生日到達判定は未実装です。
- NISAやiDeCoの運用利回り計算は未実装です。
- クラウド保存は未実装です。
- ログインは未実装です。
- 外部API連携は未実装です。
- データは同じ端末・同じブラウザ内に保存されます。
- ブラウザデータ削除や端末変更でデータが失われる可能性があります。
- 重要な入力後はJSONエクスポートでバックアップを取ってください。
